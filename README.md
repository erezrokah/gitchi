# Gitchi

Turns GitHub Pull Request Comments into a Chat

![alt Demo](Demo.gif 'Demo')

## Prerequisites

- [node + npm](https://nodejs.org/) (>=16)

- [yarn](https://yarnpkg.com/en/)

## Setup

```bash
yarn install
```

## Build

```bash
yarn build
```

## Build in watch mode

```bash
yarn watch
```

## Load extension to chrome

Load `dist` directory
