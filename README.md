# github-pullrequest-chat

Turns GitHub Pull Request Comments into a Chat

## Prerequisites

- [node + npm](https://nodejs.org/) (>=10)

- [yarn](https://yarnpkg.com/en/)

## Setup

```bash
yarn install
```

## Build

```bash
yarn build
```

## Build in watch mode

```bash
yarn watch
```

## Load extension to chrome

Load `dist` directory
